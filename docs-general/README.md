## docs-general — TDA (Trading Decisions Assistant) General Specs

Это “общий бандл” спецификаций TDA, который должен быть доступен всем сервисам.

### Что здесь лежит
- `requirements_mvp.md`: верхнеуровневые требования MVP (Flows 1–3) и список flow templates.
- `user_flows.md`: пользовательское описание потоков (как продуктовая основа).
- `contracts/`: общие контракты данных (JSON Schema, соглашения путей GCS, orchestration rules).
- `diagrams/`: диаграммы (C4).
- `background/`: исторические/контекстные документы (не блокируют разработку).

### Важно
- Цель миграции: в перспективе **удалить старый `docs/` без потерь**.
- Старый `docs/` пока остаётся как есть; этот бандл — новый source of truth для “общего слоя”.
